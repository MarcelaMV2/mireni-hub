<Enunciado>
Crear un procedimiento almacenado `sp_listar_repuestos` que devuelva todos los
repuestos ordenados por nombre. Sin parámetros.
</Enunciado>

<Resolucion titulo="Resolución">

El procedimiento envuelve un `SELECT` simple dentro de un bloque `BEGIN…END`:

<Codigo archivo="sp_listar_repuestos.sql" lang="sql" codigo={`DELIMITER //
CREATE PROCEDURE sp_listar_repuestos()
BEGIN
  SELECT id, nombre, stock
  FROM REPUESTO
  ORDER BY nombre ASC;
END //
DELIMITER ;`} />

Se ejecuta con `CALL sp_listar_repuestos();`.

</Resolucion>
